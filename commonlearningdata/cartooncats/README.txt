The set of cartoon cats images(balanced), which means
that we should have equal number of cats and non-cats
images.

REQUIREMENTS:
    size:     64x64
    format:   grayscale
    prefixes: "1_" - for cat
              "0_" - for non-cat
    += 10 cat images && 10 non-cat images

NOTE: inkscape for linux works in pretty convenient
      way to draw these drawings fast

NOTE: if you have vectorized original, you are free
      to save it in ./vectorized subfolder
